# Leon Embedded Portfolio - AGENTS.md

## Project Goal

Build a premium personal portfolio for Leon Wu, an Embedded Firmware Engineer pursuing Embedded Linux, BSP and Linux device driver development.

This project is intended for recruiters, hiring managers and embedded software engineers.

The website should demonstrate engineering capability rather than marketing ability.
